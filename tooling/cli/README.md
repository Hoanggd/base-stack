## Base Stack CLI

Scaffold a Base Stack monorepo and add apps.

- **CLI**: `base-stack`
- **Node**: >= 20

## Install

```bash
npx base-stack@latest --help
```

## Commands

### init
Create a new project.

```bash
npx base-stack init
```

### add
Add an app to an existing project (run from repo root).

```bash
npx base-stack add
```
