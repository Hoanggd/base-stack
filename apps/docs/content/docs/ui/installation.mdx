## Prerequisites

- [Node.js](https://nodejs.org/) (version 18 or higher)
- [pnpm](https://pnpm.io/) package manager

## Quick Start

Create a new Base Stack application using the CLI tool:

```bash
npx base-stack@latest init
```

## Project Structure

After running the CLI, you'll get a project with:

```
my-project/
├── apps/
│   ├── docs/           # Documentation site
│   └── [your-app]/     # Your applications
├── packages/
│   ├── eslint-config/  # Shared configs
│   ├── typescript-config/
│   └── ui/            # Shared components
└── pnpm-workspace.yaml
```

## Development

Navigate to your new project and start development:

```bash
cd ./my-project
pnpm install
pnpm dev
```

## Building

To build your project for production:

```bash
pnpm build
```
