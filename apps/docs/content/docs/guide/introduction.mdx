## What is This Project?

This is a **Next.js starter kit** that provides an opinionated, production-ready foundation for building modern web applications. It's designed to give you everything you need to start building immediately, with carefully chosen tools and patterns that work together seamlessly.

## Why This starter kit?

Instead of starting from scratch and spending time configuring tools, this starter kit gives you:

- **Opinionated project structure** that scales with your team
- **Monorepo setup** for managing multiple apps and packages
- **Best-in-class tools** that are proven in production
- **Accessibility-first approach** with React Aria components
- **Default good-looking components** that you can customize or replace
- **Modern development patterns** that make your code maintainable
- **Real-world use cases** that you can adapt to your needs

## What's Next?

Ready to get started? Visit the <Link href="/docs/guide/installation">Installation Guide</Link> to set up your project, or explore the examples to see how these tools work together in real-world scenarios. Each example is production-ready and follows modern React best practices.
